# TheWing
纵观如今市面上的主流Ai软件（指商业性的Ai聊天/对话软件，以下简称ai软件）

显而易见的，我们可以发现，其通常只允许用户同时与一个ai进行类似HTTP似的阻塞式对话

也因此，鄙人有了一项渺小的宏伟计划

或许我们可以尝试开发一款能够实现真正意义上的实时ai对话软件（或能够协助完成此项任务的插件/组件）

鄙人愿为此尽些许*绵薄之力*

**关于该项目的前景**
-简单观察社会可以发现，在现代人类的日常行为模式愈发趋近于个人主义，其表现特征却又显著的区别于纯粹的个人主义（这里不详细展开说明区别在哪，这属于哲学范畴），然而显然易见的是，在心理层面上来讲，人类是极为极端的群居动物，倘若能有一个人能够与世独立生活，那么最终不可避免的就是心理问题以及心理障碍频发，从此一根筋两头堵。现有的ai软件能够在一定程度上替补在人际交往中的某个部分，但我认为其还远远不够，下一代的ai软件需要实现真正意义上的解决这个问题。-
